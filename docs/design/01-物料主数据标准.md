# 物料主数据标准设计

> 参考 SAP MM 物料主数据体系，结合建筑机电安装行业特点简化设计。

## 1. 分类层级

```
专业 (8大专业，后4个待客户确认)
├── 电气          ← 客户已明确
├── 给排水        ← 客户已明确
├── 暖通          ← 客户已明确
├── 消防          ← 客户已明确
├── ?（待确认）    ← 可能是智能化
├── ?（待确认）    ← 可能是电梯
├── ?（待确认）    ← 可能是幕墙
└── ?（待确认）    ← 可能是其他/市政/钢结构

品类 (每个专业下若干品类)
├── 电气 → 桥架、配电箱、电缆、开关、灯具 ...
├── 给排水 → 管道、阀门、水泵、卫浴洁具 ...
├── 暖通 → 风口风阀、空调设备、风管、保温材料 ...
└── 消防 → 报警设备、喷淋、防火阀、灭火器 ...

子类 (品类下细分)
├── 桥架 → 托盘式、梯架式、槽式
├── 阀门 → 蝶阀、闸阀、止回阀、球阀
└── ...
```

对应 SAP 层级映射：`物料类型(MTART) > 物料组(MATKL) > 具体物料(MATNR)`

## 2. 物料主数据字段结构

### 2.1 通用字段（所有品类共有）

| 字段 | 说明 | 来源 | 示例 |
|------|------|------|------|
| material_id | 物料编号，系统自动生成 | 系统 | MAT-EL-BRG-00001 |
| standard_name | 标准名称（贯标后） | 标准样板库 | 托盘式热浸镀锌桥架 |
| profession | 专业 | 人工分类 | 电气 |
| category | 品类 | 人工分类 | 桥架 |
| sub_category | 子类 | 人工分类 | 托盘式 |
| spec | 规格型号 | 报价单提取 | 300×150 |
| material_type | 材质 | 报价单提取 | 热浸镀锌钢板 |
| unit | 计量单位 | 标准样板库 | m |
| exec_standard | 执行标准 | 报价单/技规 | JB/T 10216-2013 |

### 2.2 采购参考信息（系统从历史数据自动计算）

| 字段 | 说明 | 计算方式 |
|------|------|---------|
| ref_price_low | 参考价格下限 | 历史价格 P10（第10百分位） |
| ref_price_avg | 参考均价 | 历史价格均值 |
| ref_price_median | 参考中位价 | 历史价格中位数 |
| ref_price_high | 参考价格上限 | 历史价格 P90（第90百分位） |
| price_cv | 价格变异系数 | 标准差 / 均值 |
| deviation_threshold | 偏差容差(%) | 基于 CV 自动设定，可人工覆盖 |
| recommended_brands | 推荐品牌列表 | 历史中标品牌统计 |
| supplier_count | 历史供应商数量 | 去重统计 |
| last_updated | 最后更新时间 | 系统自动 |

### 2.3 扩展属性（按品类不同）

扩展属性以 JSON Schema 方式定义，不同品类挂载不同的属性集。

**桥架类：**

| 字段 | 说明 | 示例 |
|------|------|------|
| side_plate_thickness | 侧板厚度(mm) | 1.2 |
| bottom_plate_thickness | 底板/横档厚度(mm) | 1.2 |
| cover_plate_thickness | 盖板厚度(mm) | 1.0 |
| fire_rated | 是否含防火 | true/false |

**阀门类：**

| 字段 | 说明 | 示例 |
|------|------|------|
| nominal_diameter | 公称直径 | DN100 |
| nominal_pressure | 公称压力 | PN16 |
| connection_type | 连接方式 | 法兰/对夹/螺纹 |
| valve_type | 阀门类型 | 蝶阀/闸阀/止回阀 |

**风口风阀类：**

| 字段 | 说明 | 示例 |
|------|------|------|
| plate_thickness | 钢板厚度(mm) | 1.2 |
| fire_rating | 防火等级 | 70度/280度 |
| has_feedback | 是否带反馈信号 | true/false |

## 3. 物料编码规则

建议采用 `专业缩写-品类缩写-序号` 的编码方式：

```
格式: {PROF}-{CAT}-{SEQ}
示例: EL-BRG-00001  (电气-桥架-00001)
      WS-VLV-00032  (给排水-阀门-00032)
      HV-FAV-00005  (暖通-风口风阀-00005)
```

专业缩写表：

| 专业 | 缩写 |
|------|------|
| 电气 | EL |
| 给排水 | WS |
| 暖通 | HV |
| 消防 | FP |
| 智能化 | IT |
| 电梯 | EV |
| 幕墙 | CW |
| 其他 | OT |

## 4. 贯标匹配规则

系统需要识别不同写法指向同一物料，核心规则：

| 场景 | 示例 | 标准化结果 |
|------|------|-----------|
| 规格同义 | DN100 / 100mm / 4寸 / Φ108 | DN100 |
| 名称同义 | 热浸镀锌 / 热镀锌 / 热浸锌 | 热浸镀锌 |
| 品类同义 | 电缆桥架 / 线缆桥架 / 桥架 | 桥架 |
| 尺寸格式 | 300*150 / 300×150 / 300x150 | 300×150 |
| 类型同义 | 托盘式 / 槽式托盘 / 托盘型 | 托盘式 |

贯标匹配优先级：精确匹配 > 规则匹配 > AI语义匹配 > 人工确认
