# 比价权重设计

> 参考 SAP 供应商评估体系 (ME61)，适配建筑机电安装行业的招标比价场景。

## 1. 两层权重体系

比价权重分两个层面，解决不同问题：

- **A层：供应商综合评分** — 解决"选哪家供应商"的问题
- **B层：单项价格偏差容差** — 解决"这个报价是否异常"的问题

## 2. A层：供应商综合评分权重

### 2.1 评分维度与权重

| 维度 | 权重 | 评分方式 | 数据来源 | 说明 |
|------|------|---------|---------|------|
| **价格竞争力** | 40% | 自动 | 本次报价 vs 历史基准价 | 核心指标，价格越接近/低于基准价得分越高 |
| **历史合作** | 20% | 自动 | 历史中标次数、合作频率 | 反映供应商可靠性 |
| **报价完整度** | 15% | 自动 | 应报项数 / 实报项数 | 缺项多说明供货能力不足 |
| **品牌/标准合规** | 15% | 半自动 | 执行标准是否符合要求 | 是否满足技规要求 |
| **商务条款** | 10% | 人工 | 付款方式、交货周期、质保条件 | 无预付款、短周期加分 |

### 2.2 各维度评分规则

**价格竞争力（40%）：**

```
得分 = f(本次报价总价, 历史基准总价)

偏差率 = (本次报价 - 历史基准) / 历史基准 × 100%

偏差率 ≤ -10%    → 100 分（显著低于基准）
-10% < 偏差率 ≤ 0%   → 80 ~ 100 分（线性插值）
0% < 偏差率 ≤ 10%    → 60 ~ 80 分（线性插值）
10% < 偏差率 ≤ 20%   → 40 ~ 60 分（线性插值）
偏差率 > 20%     → 40 分以下，触发预警
```

**历史合作（20%）：**

```
历史中标次数 ≥ 5 次   → 100 分
3 ~ 4 次             → 80 分
1 ~ 2 次             → 60 分
0 次（新供应商）       → 40 分 + 新供应商风险标记
```

**报价完整度（15%）：**

```
完整度 = 实际报价项数 / 应报价项数 × 100%

100%    → 100 分
80%~99% → 按比例得分
< 80%   → 60 分以下 + 标记"报价不完整"
```

**品牌/标准合规（15%）：**

```
所有项目品牌均在推荐列表内 + 标准合规  → 100 分
部分项目品牌非推荐但标准合规            → 70 分
标准不合规                              → 40 分 + 预警
```

**商务条款（10%）：**

```
人工 1-100 打分，参考因素：
- 付款方式（无预付款加分，要求预付扣分）
- 交货周期（越短越好）
- 质保条件（质保期越长越好）
- 售后响应（本地有服务点加分）
```

### 2.3 SAP 标准权重对照

| SAP 标准维度 | SAP 权重键02 | 本系统对应维度 | 本系统权重 | 差异说明 |
|-------------|-------------|--------------|-----------|---------|
| 价格 | 33.3% | 价格竞争力 | 40% | 建筑采购更注重价格 |
| 质量 | 33.3% | 品牌/标准合规 | 15% | 通过品牌和执行标准间接评估 |
| 交货 | 13.3% | 商务条款(部分) | 10% | 合并到商务条款 |
| 一般服务 | 6.7% | 历史合作 | 20% | 用历史数据替代主观评分 |
| 外部服务 | 13.3% | 报价完整度 | 15% | 替换为更可量化的指标 |

## 3. B层：单项价格偏差容差

### 3.1 基于历史数据统计确定容差

不同品类的价格波动特征不同，容差应从历史数据中统计得出，而非人工拍定。

**计算方法：**

```
1. 收集某品类所有历史报价
2. 计算变异系数 CV = 标准差 / 均值
3. 容差阈值 = max(CV × 调整系数, 最小容差)
   - 调整系数建议取 1.5 ~ 2.0
   - 最小容差建议 5%
```

### 3.2 各品类建议容差（待数据验证）

以下为初始建议值，需在拿到客户完整历史数据后用统计方法校准：

| 品类 | 建议偏差容差 | 预警级别 | 依据 |
|------|------------|---------|------|
| 桥架（标准件） | ±10% | 超出黄色预警，超 20% 红色 | 市场成熟，规格标准化程度高 |
| 阀门 | ±15% | 超出黄色预警，超 25% 红色 | 品牌差异带来较大价差 |
| 管道（钢管、PPR） | ±12% | 超出黄色预警，超 20% 红色 | 受钢材/原材料行情影响 |
| 配电箱 | ±20% | 超出黄色预警，超 30% 红色 | 非标定制多，规格差异大 |
| 风口风阀 | ±15% | 超出黄色预警，超 25% 红色 | 有计价公式可参考 |
| 电缆 | ±8% | 超出黄色预警，超 15% 红色 | 铜价透明，价差主要来自品牌 |
| 特种设备（光触媒等） | ±25% | 超出黄色预警，超 35% 红色 | 市场不透明，供应商少 |

### 3.3 预警展示规则

```
偏差 ≤ 容差        → 绿色（正常）
容差 < 偏差 ≤ 2×容差 → 黄色预警（需关注）
偏差 > 2×容差       → 红色预警（异常，建议核查）
偏差 < -容差        → 蓝色提示（低于预期，可能存在规格/品质差异）
```

### 3.4 比价对比基准的选择

| 基准类型 | 计算方式 | 适用场景 |
|---------|---------|---------|
| 历史均价 | 所有历史记录的算术平均 | 默认基准 |
| 历史中位价 | 所有历史记录的中位数 | 存在极端值时更稳定 |
| 近期均价 | 最近 6 个月记录的均值 | 关注近期市场行情 |
| 最低历史价 | 历史最低成交价 | 谈判参考底线 |
| 上一轮中标价 | 上次同品类中标价 | 最直接的对标 |

系统默认使用「近期均价」作为对比基准，用户可切换。

## 4. 权重可配置性

所有权重和容差均应支持管理员在系统中调整：

- 供应商评分的 5 个维度权重可调（总和 = 100%）
- 每个品类的偏差容差可单独设置
- 预警级别的倍数阈值可调
- 对比基准类型可切换

初始值由系统预设 + 历史数据统计生成，后续由管理员根据业务经验微调。
