## 2. 数据清洗流程

### 2.1 原始数据问题清单（基于已有 720 条桥架数据观察）

| 问题 | 示例 | 处理方式 |
|------|------|---------|
| 序号 OCR 识别错误 | "了" 应为 "7" | 按上下文修正 |
| 价格字段格式不一致 | "38,74" 应为 "38.74"（逗号/点号混淆） | 统一为浮点数 |
| 材质描述不统一 | "热浸镀锌钢板" / "钢制" / "热浸锌" / "热镀锌" | 建立同义词映射表 |
| 规格分隔符不一致 | "300*150" / "300×150" / "300x150" | 统一为 "×" |
| 盖板厚度字段值异常 | 盖板厚度 = "工"（OCR 误识别 "1"） | 数值类型校验 + 人工复核 |
| 品牌名称不统一 | "天源华威" / "江苏华威线路设备集团有限公司" | 建立品牌标准化映射 |
| 备注中混合了付款条件 | 报价备注中包含付款比例、日期等 | 结构化提取付款条件字段 |

### 2.2 清洗步骤

```
1. 格式标准化
   - 价格字段 → float
   - 规格字段 → 统一分隔符 "×"
   - 日期字段 → ISO 格式
   - 厚度字段 → float，识别异常值

2. 名称贯标
   - 建立材质同义词表
   - 建立品牌标准化映射
   - 建立规格同义词表（DN100 = 100mm = 4寸）

3. 分类标注
   - 按专业/品类/子类三级分类标注每条记录
   - 标注是否含防火、盖板等附加属性

4. 去重与合并
   - 同一供应商、同一项目、同一规格的重复记录合并
   - 保留最终确认价格

5. 质量校验
   - 价格范围校验（单价 > 0，合价 = 单价 × 数量）
   - 厚度值合理性校验（0.5mm ~ 5mm）
   - 规格值合理性校验
```

## 3. 统计分析内容

### 3.1 价格分布分析（确定比价容差）

对每个品类+规格组合计算：

```python
# 伪代码
for each (category, spec) group:
    prices = all historical unit prices
    stats = {
        count:    len(prices),
        mean:     np.mean(prices),
        median:   np.median(prices),
        std:      np.std(prices),
        cv:       np.std(prices) / np.mean(prices),  # 变异系数
        min:      np.min(prices),
        max:      np.max(prices),
        p10:      np.percentile(prices, 10),
        p90:      np.percentile(prices, 90),
        iqr:      np.percentile(prices, 75) - np.percentile(prices, 25),
    }
    # 建议容差 = max(cv * 1.5, 0.05)
    suggested_threshold = max(stats['cv'] * 1.5, 0.05)
```

产出：**各品类×规格的价格基准表**，直接导入物料主数据。

### 3.2 供应商分析

```
对每个供应商计算：
- 报价次数（参与投标的频率）
- 中标次数与中标率
- 平均报价偏差（vs 最终中标价或历史均价）
- 报价完整度（参与报价的品类覆盖率）
- 价格竞争力排名（同品类中的价格分位数）
- 活跃项目数
```

产出：**供应商画像表**，用于供应商评分模型的历史合作维度。

### 3.3 价格趋势分析

```
对高频材料（桥架、阀门、管道等）：
- 按月/季度计算均价走势
- 识别季节性波动模式
- 识别长期趋势（涨/跌/平）
- 标记异常价格点
```

产出：**价格趋势图表**，用于系统仪表盘的"高频材料价格趋势"模块。

### 3.4 品类覆盖分析

```
统计：
- 各专业下已有数据的品类数量
- 各品类的历史记录数量
- 数据密度（哪些品类数据充足，哪些薄弱）
- 与 342 类标准样板库的覆盖率对比
```

产出：**数据覆盖率报告**，指导后续数据补充的优先级。

## 4. 分析产出物

| 产出物 | 用途 | 导入目标 |
|-------|------|---------|
| 品类×规格价格基准表 | 每个物料的参考价格区间和偏差容差 | 物料主数据的采购参考信息 |
| 供应商画像表 | 每个供应商的历史表现评分 | 供应商综合评分的历史合作维度 |
| 品类偏差容差参数表 | 每个品类的预警阈值 | 比价分析的预警规则配置 |
| 同义词映射表 | 名称/规格/材质的标准化规则 | AI 贯标匹配的规则库 |
| 品牌标准化映射表 | 品牌名称的统一规则 | 物料主数据的品牌字段 |
| 数据质量报告 | 清洗过程中发现的数据问题汇总 | 反馈给客户，改进数据录入 |

## 5. 时间计划

```
第 1 天：数据收集 + 现有 720 条桥架数据的清洗
第 2 天：桥架数据统计分析，产出价格基准表初版
第 3 天：等待客户提供其他品类数据，同步建立同义词映射表
第 4-5 天：其他品类数据清洗 + 统计分析
第 5 天：汇总所有分析产出物，确定比价参数初始值
```
