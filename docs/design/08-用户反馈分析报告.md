#### 逐条分析与决策

**F1：配电箱仅横向比价，历史数据只做保存**

客户明确配电箱后期只采用横向比价（同一询价轮次多供应商直接对比），历史数据只做保存，不做纵向基准分析。

- 原 Phase 4 计划的"配电箱按元器件 BOM 拆分"不再需要
- 配电箱不需要计算 reasonable_low 基准价
- 现有 `bid-matrix` 横向对比逻辑**已满足**此需求

| 决策 | 落地状态 |
|------|---------|
| **取消** Phase 4 "配电箱 BOM 元器件拆分" | ✅ plan + 功能设计已更新 |
| 配电箱走横向比价（现有能力已覆盖） | ✅ 无代码改动需要 |
| 历史数据照常导入保存，不算 reasonable_low | ✅ 功能设计已标注 |

**计价公式关键发现**（扬州新扬 Excel）：
- 风口风阀有**公式化定价**特点：价格 = f(W, H, 拼装数量, 机构类型)
- 涵盖 19 种产品（防火阀、排烟阀、送风口、百叶风口、散流器、消声器等）
- 按面积分档（S ≤ 1㎡ / 1-2㎡ / 2-3㎡），大尺寸需拆分
- 示例：70°防火阀 FD = ((W+H)*178 + W*H*178 + 35.6 + 66.75) * 拼装数量
- 注意：此价格仅针对华泾镇项目，其他项目可能有差异
