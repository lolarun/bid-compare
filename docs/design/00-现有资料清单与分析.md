# 现有资料清单与分析

> 对 `docs/现有资料` 目录中所有文件的逐项分析，明确每份资料的内容、数据价值和系统用途。

## 资料背景

这批资料来自两个招标案例和多个项目的采购记录：

- **金桥地铁上盖 J9A-03 地块** — 给排水阀门采购招标（3家供应商完整投标流程）
- **前滩 21-03 地块** — 纳米光触媒空气净化装置招标（招标文件 + 清单，无报价数据）
- **多个项目的桥架采购记录** — 港汇恒隆、华泾镇D5B、无锡104地块、温州鹿城广场、张江复旦

## 一、结构化数据（Excel，可直接导入系统）

### 1. 桥架报价数据.xls

- **格式：** Excel，多 Sheet
- **数据量：** 720 条记录
- **内容：** 5 个项目的桥架**中标采购价**（每个项目只有中标供应商一家的数据）
- **字段：** 序号、名称、规格、材质、侧板厚度、底板厚度、盖板厚度、数量、单位、本体单价（含税）、盖板单价（含税）、合价、品牌、执行标准、备注
- **备注字段中包含：** 采购日期、付款方式条款
- **各 Sheet 供应商分布：**
  - 汇总 Sheet：5 家（江苏华威、天源华威、江苏华鹏、欣浪、欣狼），实际是各项目数据拼接
  - 港汇恒隆：5 家供应商数据混合（392条）— 可能包含多家投标或分批采购
  - 华泾镇D5B：仅天源华威（207条）
  - 无锡104地块：仅天源华威（54条）
  - 温州鹿城广场四期：欣浪（36条）
  - 张江复旦：江苏华鹏（31条）
- **系统用途：** ⭐ 历史数据库种子数据 + 桥架品类价格基准计算
- **注意事项：**
  - 个别数据有 OCR 错误（序号"了"应为"7"，单价"38,74"应为"38.74"，盖板厚度"工"应为"1"）
  - 品牌名称不统一（"天源华威" vs "江苏华威线路设备集团有限公司"）

### 2. 桥架报价单格式模板.xls

- **格式：** Excel，空白模板
- **内容：** 上海建工一建标准桥架报价单模板
- **字段（16个）：** 序号、名称（备注形式）、规格、材质、侧板厚度mm、底板/横档厚度mm、盖板厚度mm、数量、单位、本体单价（含税）、盖板单价（含税）、本体及盖板合价（含税）、合计（含税）、品牌、执行标准（投标单位自填）、备注
- **系统用途：** ⭐ 定义桥架类报价单的字段解析规则

### 3. 风口风阀报价单格式.xls

- **格式：** Excel，2个有效 Sheet
- **Sheet1 - 报价单模板：** 名称、型号、规格、镀锌钢板/不锈钢板厚度、数量、单位、含税单价（13%）、价税合计、备注
- **Sheet2 - 计价公式：** 各类风阀（防火阀、排烟阀、送风口、调节阀等）的定价计算公式
- **系统用途：** ⭐ 第二种品类的报价单字段结构 + 风阀计价规则参考

### 4. 前滩21-03地块纳米光触媒空气净化装置招标清单.xlsx

- **格式：** Excel
- **数据量：** 102 条（89台纳米光触媒 + 13台光氢离子除臭装置）
- **字段：** 序号、设备名称、系统编号、风量参数、接管尺寸、单位、选型数量、含税单价、税率、金额、税额、价税合计、备注选型
- **价格字段全部为空**（待供应商填写）
- **系统用途：** 🔸 招标清单格式模板参考

### 5. 前滩21-03地块纳米光触媒招标情况表.xlsx

- **格式：** Excel
- **内容：** 一张总览表（项目名、材料类别、业主品牌要求、技规书情况、邀请投标单位、投标品牌、时间节点）
- **关键信息：** 3 家投标单位（上海百绍/上海占誉/上海梓瑾）、3 个品牌（普恩富特/爱优特/美埃）
- **系统用途：** 🔸 招标管理元数据参考

## 三、扫描件 PDF — 招标过程文件

### 9. 泰科龙第一轮商务问询函.pdf（16页）
