## 3. 各模块详细说明

### F1 工作台

#### F1.1 数据摘要卡片

| 展示项 | 数据来源 |
|--------|---------|
| 物料总数 | `materials` count |
| 供应商总数 | `suppliers` count |
| 项目总数 | `projects` count |
| 报价总数 | `quotes` count |
| 各品类统计 | 按专业分组：物料数/报价数/均价/CV/供应商数 |

#### F1.2 树状热力图

- **第一层**：可选某一时段内所有项目采购材料的总金额
- **第二层**：点开项目块可查看项目内各小类采购材料总金额
- **视觉规则**：越大越红，越小越蓝
- 可按时段筛选

#### F1.3 气泡图

- **第一层**：展示所有材料品类
  - 颜色 = 专业（电气/给排水/暖通）
  - 大小 = 采购总金额（小字备注金额）
- **第二层**：品类之下展示供应商
  - 气泡大小 = 供应商签约总金额
  - 颜色 = 品牌档位（若已划分）
  - 小字备注金额

#### F1.4 数据流 / 待办队列

- 按项目、清单分类，类似处理队列
- 清单备注状态：识别中、待审核
- 只读展示，无操作按钮

### F2 物料管理

#### F2.1 物料列表

- 分页表格：编码、名称、专业、品类、子类、规格、合理史低、参考均价、CV
- 筛选：专业、品类、子类（级联）
- 搜索：名称/编码/规格关键词
- 操作：新增、编辑、删除、查看详情
- **编码说明**：一建正在编写自有编码体系；系统先搭框架，编号留白，后续对接

#### F2.2 物料分类树

- 左侧树形结构：专业 → 品类 → 子类
- 点击节点过滤右侧物料列表
- 每个节点显示计数

#### F2.3 物料名称标准化

| 类型 | 示例 | 标准形式 |
|------|------|---------|
| 规格同义 | DN100 / 100mm / 4寸 / Φ108 | DN100 |
| 名称同义 | 热浸镀锌 / 热镀锌 | 热浸镀锌 |
| 品类同义 | 电缆桥架 / 线缆桥架 | 桥架 |
| 尺寸格式 | 300*150 / 300×150 | 300×150 |

#### F2.4 扩展属性管理

- 每个品类有不同的扩展属性（Layer 2）
- 属性分两类：
  - **[匹配]**：必须相同才可比（如规格、材质、DN）
  - **[差异]**：允许不同需说明（如厚度、表面处理）
- 存储为 JSON，前端按品类动态渲染表单
- 一建正在逐条比对确认各品类的匹配/差异划分

### F3 供应商管理

#### F3.1 供应商列表

- 字段：名称、简称、联系人、电话、经营品类、中标次数、合作评分
- 新供应商标记（win_count = 0 → 风险标识）

#### F3.2 供应商评分（5维模型）

| 维度 | 权重 | 计算方式 |
|------|------|---------|
| 价格竞争力 | 40% | 报价偏差率映射到 0-100 |
| 历史合作 | 20% | 中标次数映射 |
| 报价完整度 | 15% | 有效报价数 / 应报数 |
| 品牌合规 | 15% | 品牌档位命中率 |
| 商务条款 | 10% | 人工录入 1-100 |

#### F3.3 多供应商对比

- 选择 2-5 家供应商
- 同一品类或项目范围
- 并排展示：价格水平、评分雷达图、历史记录

### F4 项目管理

- 状态流转：进行中 → 已完成 / 已暂停
- 关联报价记录

### F5 报价管理

#### F5.1 报价记录

- 字段：物料、供应商、项目、含税单价、不含税单价、税率、数量、品牌、品牌档位、偏差率、偏差色标
- 创建报价时自动计算偏差率（与合理史低对比）
- 偏差色标：
  - 5% 以内 → 无色（正常）
  - 5%-10% → 黄色（需关注）
  - \>10% → 红色（异常偏高）
  - 阈值可在系统设置中调整
- **无报警值概念**，只有偏差值 + 色标

#### F5.2 批量导入

- 支持 Excel（.xlsx/.xls）和 CSV 上传
- 按品类选择解析模板
- 解析后预览 → 确认导入
- 自动创建不存在的物料和供应商

#### F5.3 品牌档位管理

- **品牌统一参与比对**，作为匹配项
- 只区分品牌档位（如：一档/二档/三档），不区分具体品牌
- 一建提供品牌档位初稿
- 导入时若出现**首次遇到的品牌**，弹窗要求用户写入档位
- 档位映射表可在系统设置中维护

### F6 招标比价分析（核心）

#### F6.1 横向对比矩阵（核心页面）

本次招标项目的多供应商横向比价视图。

**顶部摘要**：
- 总材料数、投标供应商数、推荐主供、最优组合总价、异常项数

**矩阵结构**：

| 列 | 说明 |
|----|------|
| 材料 | 物料名称 + 规格描述 |
| 历史均价 | 历史数据的 IQR 过滤后均价，附带**时间范围**和**项目来源** |
| 合理史低 | Q1-1.5IQR 以上最小值（主基准），附带**时间**和**项目** |
| 供应商A~N | 各供应商报价单价、偏差率（与合理史低比）、总价 |
| 最低偏差 | 所有供应商中偏差率最小值 |
| 推荐 | 推荐供应商字母标记 |

**单元格规则**：
- 绿色底色：该项最低价
- 灰色底色：未报价
- 绿色「最低」徽标：标记最低报价
- 偏差率色标：5-10% 黄色，>10% 红色，阈值可调

**底部汇总**：
- 总价（已报材料合计）
- 平均偏差

**操作**：
- 筛选（品类/子类）
- 导出矩阵

#### F6.2 单项价格比对

输入：品类、子类（可选）、新报价金额
输出：合理史低、历史均价、P10-P90、偏差率、色标

**比价基准定义**（用户反馈确认）：

| 基准 | 计算方式 | 用途 |
|------|---------|------|
| **合理史低**（主基准） | IQR 过滤后最小值 = min(prices where price >= Q1 - 1.5*IQR) | 偏差率计算 |
| 历史均价 | IQR 过滤后算术均值 | 辅助参考 |
| 最低历史价 | 所有历史数据绝对最低 | 提醒事项（保留但不作为基准） |

**偏差率计算**：
```
deviation_pct = (new_price - 合理史低) / 合理史低 × 100%
```

**色标判定**（不再用 green/yellow/red/blue 报警）：
```
|deviation| ≤ 5%   → 无色（正常）
5% < |deviation| ≤ 10% → 黄色（需关注）
|deviation| > 10%  → 红色（异常）
阈值可在系统设置中按品类单独配置
```

#### F6.3 品类统计分析

- 各子类价格分布（数据量/均价/中位价/标准差/CV/P10/P90）
- 条件格式：CV < 0.5 绿、0.5-1.0 黄、> 1.0 红
- ~~异常价格检测（Modified Z-score > 3.0）~~ → **已取消**，不再使用修正 Z 分

#### F6.4 比价报告导出

生成标准 Excel 报告，包含横向对比矩阵、品类统计等。

### F7 系统设置

#### F7.1 评分权重配置
- 5 维合计 = 100%，可调

#### F7.2 偏差阈值配置
- 各品类黄色/红色阈值可单独设置
- 默认：黄色 5%，红色 10%

#### F7.3 品牌档位映射表
- 品牌 → 档位（一档/二档/三档）
- 一建提供初稿，系统可维护
- 新品牌导入时弹窗写入

### F8 数据导入导出

#### F8.1 Excel 批量导入

10 个品类的解析模板：

| 品类 | 关键列 |
|------|--------|
| 桥架 | 名称、规格、材质、厚度×3、单价、品牌 |
| 阀门 | 名称、规格、型号、材质×5、价税合计、品牌 |
| 风口风阀 | 名称、型号、规格、钢板厚度、含税单价、品牌 |
| 母线槽 | 名称、母线类型、规格型号、铜牌厚度、含税单价、品牌 |
| 配电箱 | 配电箱编号、配置尺寸、配置品牌（盘柜厂）、主进线规格、单价（按整箱横向比价） |
| 不锈钢管 | 名称、规格、壁厚、牌号、含税单价、品牌 |
| 水箱 | 名称、规格型号、价税合计、品牌 |
| 潜水泵 | 名称、型号、流量/扬程/功率、单价、品牌 |
| 风机盘管 | 名称、型号、管制、风量、单价合计、品牌 |
| 空调泵 | 名称、规格、流量/扬程/功率、单价、品牌 |

#### 配电箱特殊处理（2026-05-20 更新）

- ~~不按整体比对，按元器件 BOM 拆分比较~~ → **已取消**
- **仅采用横向比价**：同一询价轮次中多供应商的配电箱报价直接对比
- 历史数据**只做保存**，不参与纵向基准（reasonable_low）分析
- 品牌字段含义：**盘柜制造厂商**品牌（非元器件品牌如施耐德/ABB）
- 品牌档位对配电箱品类时，评价的是盘柜厂的档位

#### 数据稀疏品类说明（2026-05-20 确认）

以下品类历史数据不足，统计分析结果仅供参考：

| 品类 | 数据行 | 供应商数 | 说明 |
|------|--------|----------|------|
| 水箱 | 137 | 多家 | 客户确认数据少，暂无补充计划 |
| 空调泵 | 128 | 多家 | 客户确认数据少，暂无补充计划 |

## 4. 用户反馈变更摘要

| # | 变更项 | 旧方案 | 新方案（一建确认） |
|---|--------|--------|-------------------|
| 1 | 比价基准 | 历史均价/中位价 | **合理史低**（Q1-1.5IQR 以上最小值） |
| 2 | 报警机制 | 四级报警 green/yellow/red/blue | **仅偏差值 + 色标**（黄色 5-10%，红色 >10%，阈值可调） |
| 3 | 异常值检测 | 修正 Z 分 > 3.0 | **取消**，不再使用 |
| 4 | 最低历史价 | 不单独展示 | **保留**在对比表中作为提醒，但不作为基准 |
| 5 | 品牌比价 | 品牌仅作筛选/评分 | 品牌作为**匹配项**，按**档位**区分（一档/二档/三档），不区分具体品牌 |
| 6 | 品牌档位录入 | 无 | 导入时**首次品牌弹窗**写入档位 |
| 7 | 配电箱比价 | 按整箱比价 | ~~按元器件级别比较~~ → **仅横向比价，历史数据只做保存**（2026-05-20 梁老师反馈取消 BOM 拆分） |
| 8 | 备注信息 | LLM 提取付款/质保/技术 | 付款、质保、技术补充**不参与比对**，仅在结果中以备注形式展现 |
| 9 | 物料编码 | 系统自动编码 | 先搭框架**编号留白**，待一建提供自有编码 |
| 10 | 横向对比矩阵 | 无此页面 | **新增核心页面**，历史均价+合理史低列含时间/项目信息 |
| 11 | 工作台可视化 | 仅统计卡片 | 新增**树状热力图**、**气泡图**、**数据流/待办** |
