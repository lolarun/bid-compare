### v2 主要变更（2025-05-19）

基于一建用户反馈做了以下关键调整：

- **比价基准**：从历史均价/中位价改为「合理史低」（IQR 过滤后最小值）
- **偏差机制**：取消四级报警（green/yellow/red/blue），改为偏差值 + 色标（黄色 5-10%，红色 >10%）
- **异常值检测**：取消修正 Z 分（Modified Z-score）
- **品牌维度**：品牌作为匹配项参与比对，按档位（一/二/三档）区分，不区分具体品牌
- **配电箱**：不按整箱比对，按元器件级别拆分比较
- **核心页面**：新增「横向对比矩阵」（招标比价分析的主交互界面）
- **工作台**：新增树状热力图、气泡图、数据流/待办
